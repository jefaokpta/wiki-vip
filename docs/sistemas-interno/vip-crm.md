# VIP-CRM

---

O sistema Vip-CRM foi desenvolvido para utilização do **Departamento de Implantação**. Ele possui um formulário amplo, que contém todas as infomrações necessárias na chegada de um novo cliente, e uma vez que este formulário for preenchido, permite que nossa implantação tome duas ações a partir de um simples **clique**:

- **Cadastro automático da empresa no servidor Vip**: Preenche automaticamento o cadastro da empresa no servidor escolhido, gerando o código da empresa e as configurações inicias no Sistema Vip.
- **Geração automática de tickets**: cria todos os tickets necessários, para cada departamento envolvido no processo de implantação da nova empresa **(Financerio, Suporte Técnico e Tráfego)**.
