# TRAFFIC-VIP

---

O sistema LicenciadosTraffic-Vip foi desenvolvido para utilização do **Departamento de Tráfego e Planejamento**. Ele é responsável pela gestão de cada **DDR** disponibilizada e comercializada pela Vip.

Nele, é possível verificar as **DDR's Disponíveis**, encontrar todas as **DDR's Vinculadas** e utilizadas por uma determinada empresa, **desvincular uma DDR** de uma empresa e também **cadastrar novos números** a ser comercializados pela Vip.
