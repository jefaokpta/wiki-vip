# LICENCIADOS

---

O sistema Licenciados foi desenvolvido para utilização do **Departamento Finaceiro**. Ele permite que cada servidor licenciado seja cadastrado nele, assim como os valores de suas respectivas tarifas **(cadência, fração e valor)**. Cada centro de custo utilizado no servidor Jupiter deve ser cadastrado para cada licenciado, possibilitando assim, que realizemos o processo correto de tarifação do mesmo.

Este sistema busca no servidor Jupiter, todas as madrugadas, os dados de todas as chamadas registradas, e sendo assim, trabalha sempre com dados do dia anterior **(D-1)**.
