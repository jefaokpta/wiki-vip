#### ACESSO AS GRAVAÇÕES DO VIP

A partir do dia **19/06/2026**, tivemos uma alteração na forma como disponibilizamos o acesso as gravações das chamadas no sistema Vip. Inicialmente, quando um usuário clicava para ouvir um gravação, o sistema Vip gerava um link de acesso diretamente ao arquivo do cliente (arquivo-estático), e uma vez que o cliente obtivesse este link, ele conseguia ter acesso a gravação, estando ou não logado no sistema.

Mas a partir da data mencionada, este acesso foi modificado: **agora o cliente só tem acesso a gravação se estiver com uma sessão ativa no sistema Vip (logado)**, de forma que mesmo possuindo o link correto da gravação, se não estiver logado, o acesso não fica mais disponível.

> **Obs:** Qualquer usuário com acesso a visualização dos relatórios, poderá (se assim quiser), baixar qualquer gravação **(download)**, afinal, o player de execução do áudio permite isso. O mais importante neste caso, é que este nível de acesso só seja concedido a usuários que realmente necessitam dele.

#### EXTRAÇÃO DE RELATÓRIOS

Os servidores compartilhados da Vip **(CallCenter, IPBX e IPBCenter)** podem apresentar restrições de no período de extração de alguns relatórios. Isso acontece para evitar que o sistema fique indisponível caso um cliente resolvar solicitar grandes períodos, e com isso, interfira na qualidade das chamadas e também nos acessos de outros clientes do mesmo servidor.

Por isso, caso a empresa possua mais de 20 agentes, o sistema pode impedir que relatórios como Estatísticas dos Agentes, fiquem restritos ao período semanal (7 dias).

O relatório de chamadas no período pode sofrer restrições para buscas mensais (período de 30 dias).

> **Obs:** Estas regras não costumam ser aplicadas na empresa de **código (100)**, que é a própria Vip Solutions.
