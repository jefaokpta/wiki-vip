# VIP Worker (O Processador de Chamadas)

É o "operário" que trabalha nos bastidores no exato momento em que uma ligação está acontecendo. Ele é o responsável por atender a chamada, tocar a música de espera, direcionar o cliente pelas opções digitadas no menu (URA) ou colocá-lo em uma fila de atendimento para falar com um atendente. Quando a ligação termina, ele avisa o sistema central sobre a duração da conversa para que os relatórios sejam gerados.
