# VIP PABX Manager (O Gerenciador Central)

É o "cérebro" administrativo da plataforma de telefonia. Este sistema não faz as ligações em si, mas guarda todas as regras do negócio. É nele que ficam cadastrados os ramais, as filas de atendimento, os menus de voz (URA), os horários de funcionamento e os usuários. Além disso, ele gerencia as permissões de acesso e é responsável por calcular os custos das ligações e processar os relatórios que vemos na tela.
